Defense Manpower Data Center (DMDC) Training Games

Web Development

Project Overview

Challenge:

Flash games are becoming difficult to run on various browsers and are terribly outdated.

Solution:

Converting two legacy flash-based training games to HTML5 for confidential government client.

Role:

Interactive Web Development, Content Strategy

Tool:

HTML5, CSS3, Javascript, AWS

The Results

Because the games were designed and created 7+ years ago, the original artwork looked outdated and had to be completely recreated by graphic designers. The content of the training was kept the same besides some minor content changes I made. My job was to add required interactions to all the graphics for all three levels of the two games using HTML5 and Javascript.

A few examples of interaction I added:

  • Hovering tooltips when selecting items from the main menu,
  • Added SVG outline overlays for representing eligible areas for drag and drop on the screen,
  • Adding clickable hotspot regions on graphics
  • Providing feedback to the user (green check mark/red "X" with helpful text if correct/incorrect)
